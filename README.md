# Landing - Festival Gastronómico de Vanguardia

Pequeña landing estática creada para el festival. Incluye HTML, CSS y JavaScript para un formulario de reservas (frontend solamente).

Archivos:
- `landing.html` - página principal.
- `styles.css` - estilos responsivos.
- `script.js` - validación simple y simulación de envío del formulario.

Cómo probar localmente:
1. Abre `landing.html` en cualquier navegador moderno.
2. Rellena el formulario en la sección "Entradas" y haz click en "Reservar".

Siguientes mejoras recomendadas:
- Conectar el formulario a un endpoint real (por ejemplo, un servidor con Node/Express o un servicio de formularios).
- Añadir imágenes reales y optimizadas.
- Mejorar la accesibilidad y pruebas de contraste.
- Preparar despliegue en Netlify o GitHub Pages.
